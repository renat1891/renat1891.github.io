## 📋 Опис завдання

Вам потрібно створити вебдодаток, у якому користувачі можуть читати відгуки інших людей та залишати свої власні.

### Функціональні вимоги:
1.  **Головна сторінка (`/`)**:
    *   Відображає список усіх відгуків.
    *   Кожен відгук повинен містити: **Ім'я автора**, **Текст відгуку** та **Оцінку** (від 1 до 5 зірочок/балів).
    *   Під списком (або над ним) розміщується HTML-форма для додавання нового відгуку.
2.  **Форма додавання відгуку**:
    *   Поле для введення імені (текстове).
    *   Поле для тексту відгуку (текстове або textarea).
    *   Вибір оцінки від 1 до 5 (можна використати `<select>` або звичайне числове поле `<input type="number">`).
    *   Кнопка «Надіслати».
3.  **Обробка форми (`POST` запит)**:
    *   Дані з форми мають надсилатися на сервер (наприклад, на той самий маршрут `/` або окремий `/add`).
    *   Сервер має отримати ці дані, створити новий відгук та зберегти його.
    *   *Підказка:* Оскільки ми ще не вивчали бази даних, зберігайте відгуки у звичайному списку Python (глобальній змінній), наприклад:
        ```python
        reviews = [
            {"name": "Олексій", "text": "Чудовий сервіс!", "rating": 5},
            {"name": "Марія", "text": "Непогано, але можна краще.", "rating": 4}
        ]
        ```

## 💡 Корисні підказки та поради

### 1. Перенаправлення (Redirect)
Після того як користувач надіслав форму (POST-запит) і ви успішно додали відгук до списку, найкраща практика — перенаправити користувача назад на головну сторінку за допомогою `redirect` та `url_for`. Це запобігає повторному надсиланню даних при оновленні сторінки (F5).

Імпортуйте їх із Flask:
```python
from flask import Flask, render_template, request, redirect, url_for
```
Приклад перенаправлення на функцію головної сторінки:
```python
return redirect(url_for('index')) # де 'index' — це назва функції для маршруту '/'
```

### 2. Структура проєкту
Створіть окрему папку для проєкту (наприклад, `guestbook/`), у якій будуть:
*   `app.py` — логіка додатка.
*   папка `templates/` з файлом `index.html`.

## 🛠️ Як перевірити роботу?
1. Запустіть сервер (`python app.py`).
2. Відкрийте браузер, перейдіть на сторінку.
3. Спробуйте додати відгук. Переконайтеся, що він з'явився у списку.
4. Спробуйте видалити будь-який відгук і перевірте, чи зник він зі списку.
